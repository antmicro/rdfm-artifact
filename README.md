RDFM Artifacts Library
======================

RDFM - remote device fleet management and OTA update system with incremental update capabilities utilizing binary deltas, designed for embedded Linux systems.

See https://github.com/antmicro/tegra-demo-distro-with-rdfm for an example usage.

This repo contains sources for the `rdfm-artifact` command-line tool, which is used for creating and modifying artifacts.
See `rdfm-artifact --help` for example usage.

Note: this software is based on the Mender OTA update system, but is not affiliated with it in any way.
